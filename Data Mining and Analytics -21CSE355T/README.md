## Subject Overview

This course, "Data Mining and Analytics," is a professional elective offered by the School of Computing. It focuses on extracting useful patterns and knowledge from large datasets using various techniques and algorithms. The course covers essential topics like data preprocessing, association rules, classification, clustering, and outlier detection. It is structured as 2 lectures, 1 tutorial, and 0 practicals per week, totaling 3 credits, with no prerequisites, co-requisites, or progressive courses.

The course is divided into five units, each spanning 9 hours:
- **Unit 1: Data Mining Introduction** – Covers data types, patterns, visualization, and preprocessing (cleaning, integration, transformation, discretization, reduction).
- **Unit 2: Associations and Correlations** – Explores market basket analysis, Apriori algorithm, frequent itemsets, multilevel and multidimensional rules, and correlation analysis.
- **Unit 3: Classification and Prediction** – Discusses decision trees (ID3, CART), Bayes methods, SVM, KNN, backpropagation, performance metrics, and regression.
- **Unit 4: Cluster Analysis** – Includes partitioning, hierarchical methods (agglomerative/divisive), probabilistic models, and techniques like BIRCH, DBSCAN, STING, CLIQUE.
- **Unit 5: Outliers and Statistical Approaches** – Addresses outlier detection (supervised/unsupervised), challenges, and applications in recommender systems, intrusion detection, and financial analysis.

By the end, you'll gain skills in pattern discovery, algorithm application, and using data mining tools for real-world problems.

## Real-World Projects and Applications

Data mining isn't just theoretical—it's applied everywhere to solve practical problems. Here are some real-world applications and project ideas inspired by industry examples. These can be great for your portfolio or mini-projects during the course:

### Applications in Various Sectors
- **Retail (e.g., Walmart, Amazon)**: Uses market basket analysis to recommend products and optimize inventory. Walmart analyzes 2.5 petabytes of data hourly for personalized shopping and supply chain efficiency.
- **Healthcare**: Predictive modeling for patient outcomes, fraud detection in insurance claims, and personalized treatments. For example, analyzing electronic health records to predict disease risks.
- **Finance and Banking**: Fraud detection in transactions, credit scoring, and financial forecasting. Banks like JPMorgan use it to spot anomalies in real-time.
- **Education**: Predicting student performance or identifying top schools based on test data. NYC schools use it to analyze standardized tests for improvement strategies.
- **Marketing and Advertising**: Personalized ads on platforms like Netflix or Spotify, which mine user behavior for recommendations.
- **Telecommunications**: Enhancing network performance and customer service by analyzing usage patterns.
- **Energy Sector**: Predicting outages and optimizing resource allocation, as seen in utility companies reducing downtime.
- **Intrusion Detection**: Cybersecurity firms mine network logs to detect threats.
- **Biological Data**: Analyzing genomic data for drug discovery or evolutionary patterns.
- **Location-Based Services (e.g., Starbucks)**: Using demographics and traffic data to choose optimal store locations.

### Project Ideas for Students
These align with the course units and can be implemented using tools like Python (with libraries such as scikit-learn, pandas) or WEKA for data mining:
1. **Market Basket Analysis (Unit 2)**: Analyze a retail dataset (e.g., from Kaggle) using Apriori algorithm to find product associations. Real-world: Build a simple recommender for an e-commerce site like suggesting "If buy bread, also buy butter."
2. **Student Performance Prediction (Units 1 & 3)**: Use classification algorithms (e.g., Decision Trees or KNN) on educational data to predict grades. Real-world: Similar to systems in online learning platforms like Coursera.
3. **Fraud Detection in Finance (Units 3 & 5)**: Apply SVM or outlier detection on credit card transaction data to flag suspicious activities. Real-world: Inspired by banking apps that alert for unusual purchases.
4. **Customer Segmentation (Unit 4)**: Cluster customers based on shopping habits using K-Means or DBSCAN. Real-world: Retail giants like Target use this for targeted marketing (e.g., predicting pregnancies from purchase patterns).
5. **Sentiment Analysis on Social Media (Units 1 & 3)**: Mine Twitter data for opinions on products using Naive Bayes. Real-world: Brands monitor feedback to improve products.
6. **Outlier Detection in Healthcare (Unit 5)**: Identify anomalies in patient vital signs data. Real-world: Hospitals use this for early disease detection.
7. **Web Scraping and Exploratory Analysis (Unit 1)**: Scrape job portals or Wikipedia, clean the data, and visualize trends (e.g., university preferences). Real-world: Analysts use this for market research.
8. **Recommendation System (Unit 5)**: Build a simple movie recommender using correlation analysis. Real-world: Netflix's engine, which drives 80% of views.

Start with datasets from Kaggle or UCI Machine Learning Repository. These projects will help you apply concepts and build a strong portfolio for jobs.
